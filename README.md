# GoToGram
* fully vibecoded 

Send media from X (Twitter) to Telegram channels in one click.

GoToGram is a Chromium extension that adds a Telegram sharing button to media on X (formerly Twitter), allowing you to quickly publish content to your Telegram channel using your own bot.

## Installation

### Clone the repository

```bash
git clone https://github.com/yourusername/gotogram.git
cd gotogram
```

### Install dependencies

```bash
npm install
```

### Run the website locally

```bash
npm run dev
```

## Permissions

GoToGram requires access to:

* `x.com`
* `twitter.com`
* `api.telegram.org`
